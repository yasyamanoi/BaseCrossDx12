# BaseCrossDx12
DirectX12用のBaseCrossフレームワークです  
BaseCrossフレームワークはゲーム作成用のフレームワークです。  
Dx11版は、以下を参照ください。  
https://github.com/WiZFramework/BaseCross64  

DirectX12用のフレームワークは、まだ作成をはじめたばかりです。  
前バージョンがそうであったように、初期段階から公開する事としました。  
ですのでフレームワークというには、まだまだ程遠い内容となっております。  

長い目で見ていただければ幸いです。
